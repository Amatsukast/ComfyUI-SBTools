# ComfyUI Manager Registration Files

This folder contains files for registering ComfyUI-SBTools with ComfyUI Manager.

## Files

- **`custom-node-list-entry.json`** - Entry to be added to ComfyUI-Manager's `custom-node-list.json`
- **`SUBMISSION_GUIDE.md`** - Step-by-step guide for submitting the Pull Request

## Quick Start

1. Read `SUBMISSION_GUIDE.md` for detailed instructions
2. Fork https://github.com/ltdrdata/ComfyUI-Manager
3. Add the content from `custom-node-list-entry.json` to `custom-node-list.json`
4. Submit a Pull Request

## Note

This is a **one-time process**. After the PR is merged, updates to ComfyUI-SBTools will automatically be available to users via ComfyUI Manager's update feature.
